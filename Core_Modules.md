# Core Modules

## 1. Digital Glossary Module

The primary component of the system that allows users to search, browse, and explore Kalibugan vocabulary with translations in Filipino and English.

### Features

- Search by Kalibugan word
- Search by Filipino word
- Search by English word
- Smart search with autocomplete
- Alphabetical (A–Z) browsing
- Browse by categories
- Advanced search filters
- Recently searched words
- Most searched words
- Favorites / Bookmarks
- Offline glossary access
- Search history

---

## 2. Multilingual Dictionary Module

Provides complete lexical information for every documented word.

### Each Entry Includes

- Kalibugan word
- Filipino translation
- English translation
- Pronunciation guide
- Native audio pronunciation
- Part of speech
- Definition
- Example sentence
- Sentence translation
- Synonyms
- Antonyms (if available)
- Word origin (Etymology)
- Alternate spellings
- Cultural notes
- Images
- Video demonstration (optional)

---
